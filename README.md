# php-flat-file-guestbook
Flat-file guestbook written in php
# Features
* no database rquired
* block IP adressess
* possibility to approve messages before posting
* bad words filter
* max characters options 
* email option
* TinyMCE texteditor
* Bootstrap 4 grid
